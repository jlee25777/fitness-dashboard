# Interactive Fitness Dashboard

## Overview
An interactive dashboard analyzing fitness data to answer the question: how can you burn the most calories? Built with Python Dash and Plotly with real-time filtering controls.

## Features
- BMI distribution histogram with range slider filter
- Average calories burned by workout type with dropdown filter
- Calories burned by workout frequency with box plot
- BMI distribution by calorie burn category with violin plot
- Calories burned by gender and workout type with grouped box plot

## Tech Stack
- Python
- Dash
- Plotly
- Pandas

## Key Findings
- HIIT burns the most calories per session (~1,650) followed by Strength, Cardio, and Yoga
- Higher workout frequency correlates with increased calorie burn
- BMI distribution is similar across calorie burn categories suggesting workout type matters more than body size
